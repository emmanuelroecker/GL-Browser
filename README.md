# GLICER Browser

## Features

### Requests blocker

filter.yml in match pattern format

### Customize the way a web page displays or behaves

config.yml -> cfg directory
Inject CSS/JS

## Install & Start Browser

[node.js](https://nodejs.org/) must be installed.

```console
npm install
npm start
```
